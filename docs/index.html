Ads free DNS-over-HTTPS
iOS.mobileconfig
  1. Go to Settings > Network & internet
  2. Select Advanced > Private DNS
  3. Select the Private DNS provider hostname option
  4. Enter server name: adsfree.fly.dev
  5. Or ip addresses: 66.241.125.228, 2a09:8280:1::42:b246 if required
In Firefox open about:preferences#privacy go to Enable DNS over HTTPS section.
Select Max Protection option and Custom provider.
Then put https://adsfree.fly.dev url into the textbox.
